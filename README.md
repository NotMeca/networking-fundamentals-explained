# Networking Fundamentals Explained

This repository documents my practical understanding of core networking concepts required for SOC-level cybersecurity roles.

## Topics Covered

- OSI vs TCP/IP model (practical comparison)
- TCP 3-way handshake
- DNS resolution process
- Common ports (22, 80, 443, 3389, 53)
- What happens when a user accesses a website

## Tools Used

- Cisco Packet Tracer
- Wireshark
- Windows OS

## Objective

To build a clear and structured understanding of how network traffic flows in real environments and how it relates to security monitoring.
